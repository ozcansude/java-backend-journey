

# PostgreSQL Employee Analytics Practice

> Bu projede amacım, SQLBolt'tan öğrendiğim konuları hazır aldığım verilerle üzerinde düzenleme ve manipülasyon işlemleri yaparak pekiştirmek.

Odak konular:

- SELECT
- WHERE
- ORDER BY
- LIMIT
- JOIN
- GROUP BY
- HAVING
- temel aggregate fonksiyonları: COUNT, SUM, AVG, MIN, MAX

## Dosyalar

- schema.sql: tabloları oluşturur
- insert_employees.sql: örnek verileri ekler
- exercises.md: çözülmesi gereken görevler
- queries.sql: çözümlerini yazacağım dosya
