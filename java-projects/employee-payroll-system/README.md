# Employee Payroll Management System

Bu küçük Java projesini nesne yönelimli programlamayı daha iyi öğrenmek için oluşturdum.

## Bu Projede Neler Çalışacağım

* Class ve object kullanımı
* Inheritance yani kalıtım
* Abstract class kullanımı
* Interface kullanımı
* Method override etme
* ArrayList ile veri tutma
* Try-catch ile hata yakalama
* Kendi exception sınıflarımı oluşturma
* Küçük bir Java projesini dosyalara ayırarak düzenleme

## Proje Fikri

Bu proje basit bir çalışan maaş yönetim sistemidir.

Projede farklı çalışan türleri oluşturuyorum ve her çalışan türünün maaşı farklı şekilde hesaplanıyor.

Çalışan türleri:

* Full-time employee
* Part-time employee
* Contract employee

Her çalışan tipi kendi maaş hesaplama mantığına sahip.

## Bu Projeyi Neden Yaptım

Bu projeyi Java temellerimi geliştirmek ve nesne yönelimli programlamayı daha iyi anlamak için yaptım.

Ayrıca GitHub profilimde backend öğrenme sürecimi gösterebilecek küçük ama düzenli bir proje olmasını istedim.
